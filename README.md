# Raycast Local Files Search Extension

A Raycast extension that provides fuzzy search capabilities for files in your Downloads folder, powered by a local MCP (Model Context Protocol) server.

## Features
- Fuzzy search for files in Downloads folder
- Real-time search results
- File type detection and display
- Error handling and loading states

## Setup
1. Install dependencies: \`npm install\`
2. Start the MCP server
3. Run the extension: \`npm run dev\`

## Development
- Built with Raycast API
- Uses Express.js for the MCP server
- TypeScript implementation
